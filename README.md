# Secure Hybrid Transformer Network for Predictive Maintenance

## Overview
This project implements an AI-powered predictive maintenance system designed to forecast failures in industrial electrical equipment. The system analyzes time-series sensor data such as **voltage, current, and temperature** using a Transformer-based neural network.

To enhance system security, the project also simulates cyber attacks and applies **AES encryption** to secure sensor data transmission.

This hybrid approach combines **AI, cybersecurity, and IoT monitoring** to improve reliability and protection of industrial systems.

---

## Features
- Predicts equipment failures using machine learning
- Processes time-series sensor data
- Uses a **Transformer-based neural network** for prediction
- Simulates cyber attacks such as:
  - False data injection
  - Sensor manipulation
- Implements **AES encryption** to secure sensor data
- Evaluates system performance under different scenarios:
  - Normal data
  - Attacked data
  - Encrypted data

---

## System Workflow
1. Sensor data generation (voltage, current, temperature)
2. Data preprocessing
3. Transformer neural network training
4. Cyber attack simulation
5. AES encryption of sensor data
6. Performance evaluation under different conditions
